# ダジャレ音読

[ダジャレで音読の練習をするゲーム](https://marmooo.github.io/dajare-ondoku/)です。

## Installation

- install [SudachiDict](https://github.com/WorksApplications/SudachiDict)

## Build

```
deno run --allow-read --allow-write --unstable build.js
deno test --allow-read
bash build.sh
```
